Monthly reprice run. Anchored on the fixed spec (AM5, 16c/32t, GPU value rule, 64GB DDR5 on a
4-DIMM board, bare-metal Linux host + isolated KVM worker VM). Tweakers Pricewatch was unreachable

## Headline: the GPU pick didn't move, the RAM/storage lines exploded

RX 7900 XTX 24GB remains the pick under the value rule — nothing in this run's data changes that.
The real story is a structural AI-driven DRAM + NAND shortage that pushed the 2×32GB DDR5-6000 CL30
kit from a ~€200–270 estimate to a confirmed **€959–1,079**, and 1TB+2TB NVMe storage from ~€180 to
**€649**. Both are corroborated by independent industry reporting (TrendForce, IDC, Tom's Hardware),
not shop-specific noise, and neither is expected to ease before 2027.

## Changed vs Run 002

- **RAM (the volatile line, as flagged):** confirmed live at €959–1,079 for the 2×32 DDR5-6000 CL30
  kit — ~4× the Run 002 estimate. Root cause: AI/HBM fab reallocation squeezing standard DDR5
  supply (Samsung/SK Hynix diverting capacity to HBM). Industry consensus: no relief before 2027,
  possible further increases through end of 2026.
- **Storage, newly volatile:** same AI-driven fab-capacity squeeze hit NAND flash. 1TB+2TB NVMe now
  €649 vs the ~€180 assumed in Run 001/002 — a ~3.6× jump, not previously tracked as a risk line.
- **CPU down slightly:** €609 vs ~€649–700 in Run 002.
- **GPU pick unchanged**, price roughly flat; NL retail stock for new units patchier, EU-wide and
  used supply still fine.
- **ROCm 7.2.1** fixed the gfx1201 (RDNA4) memory-coherency bug — noted, doesn't flip the pick.
- **R9700 32GB** priced firmly now (~€1,420–1,500) instead of an estimate — still a deliberate
  escape hatch, not the default.
- **Total build ~60–75% above Run 002's ~€1,830–2,350** — almost entirely RAM + storage, not a
  compute-pick change.

## Buy-now-vs-wait verdict (RAM)

**Buy now if 64GB is genuinely needed soon.** This is a structural shortage, not a transient spike
— every tracked forecast expects DDR5 pricing to stay elevated or keep climbing through the rest of
2026, with normalization not expected before 2027. Waiting is more likely to cost more than today's
€959–1,079, not less. If budget is the blocker, the fallback is a **provisional 2×16GB (32GB) kit
now** (~a third of the price) with the jump to 64GB deferred as a later upgrade — this means
temporarily missing the 64GB hard constraint, so it's presented as a tradeoff for Joost to decide,
not resolved here. Storage: same logic but lower stakes — 1TB alone (€230) covers the boot drive;
the 2TB bulk/VM drive can be deferred without touching a hard constraint.
